藤井 風 ( Fujii Kaze )

1997年6月14日生まれ。幼少期より父の影響でクラシックピアノを始め、12歳の時に実家の喫茶店で撮影したピアノカヴァー動画をYouTubeに投稿したことが、後に音楽の世界へ飛び込むきっかけとなる。

2020年5月、1st Album「HELP EVER HURT NEVER」(ー常に助け、決して傷つけないー)リリース。 2022年3月、2nd Album「LOVE ALL SERVE ALL」(ー全てを愛し、全てに仕えよー)リリース。

藤井 風が大切にしているコアメッセージを掲げた２作品は、共にBillboard Japan総合アルバムチャート”HOT Albums”並びにオリコン週間アルバムランキングにて1位を獲得した。 2022年10月、2ndアルバムリリースを記念した「LOVE ALL SERVE ALL STADIUM LIVE」と題したスタジアムライブを、音楽ライブの開催は史上初となる大阪府吹田市のPanasonic Stadium Suitaにて開催し、二日間で７万人を動員した。 この模様は、2023年3月10日よりNetflixにて全世界配信されている。

青春病 ( SEISHN SICK )

この曲は学生時代の青春ではなく、人生における青春を謳っている。
青春の葛藤や刹那的な感情をリアルに描写し、聴く人々に共感と勇気を与えます。
メロディは心に残り、歌詞は日常の喜びや苦悩を深く掘り下げます。その独特のサウンドと落ち着く歌声、深い歌詞が、聴く者の心に響き、青春の複雑さや美しさ、儚さを感じさせてくれます。

grace

わたしはあなた、あなたはわたし。誰もが持っている「higher self」、こんな自分と向き合える歌詞がほかにあるでしょうか。
１番の歌詞のネガティブな感情が、サビの「涙も輝き始めた」のあとから徐々に明るく、しかし、どこか儚い雰囲気が漂います。
そしてこの曲の魅力の１つが、藤井風がわざとヘルツを４３２㎐にして聴く者に昭和の歌謡曲のような懐かしさや落ち着きを与えてくれます。これはぜひ実際に聞いてみてください
